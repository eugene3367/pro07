# 프로젝트
## 프로젝트 개요
"과학기술정보통신부" 라는 가상 기업을 주제로 프론트엔드 부분을 설계 시에는 피그마를 이용하여 프로토타입 사용성을 테스트 한 후, html과 css, javascript로 작성했고, 백엔드 부분은 Spring legacy를 이용하여 Controller, Model(DAO), DTO, EL, JSTL 등을 통한 데이터 바인딩을 구현했음
해당 프로젝트는 회원기능인 회원가입, 로그인, 로그아웃, 회원 정보 확인, 회원 정보 수정, 회원 탈퇴, 회원의 게시판 글 목록, 글 상세보기, 글 등록, 글 수정, 글 삭제 등의 기능을 구현하였으며, MYSQL, MyBatis, DI 및 IoC, JSON, AOP, Validation, 비밀번호 암호화(Spring security), 인터셉터, ckeditor, 트랜잭션 처리 등을 활용하여 웹 애플리케이션을 개발했음.
